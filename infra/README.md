# docker-compose 사용법
> 회사 PC 말고 집에서 사용 권장
> 1. `docker desktop` 다운로드 [docker desktop](https://www.docker.com/products/docker-desktop/)
> 
> 
>
> 2. 명령어로 폴더 이동
> ``` shell
> cd ./infra/ 
>    ```
> 3. docker compose 실행
> ``` shell
> docker-compose up -d # -d는 옵션
> # 또는
> docker compose up -d # -d는 옵션
> ```
> 4. docker compose 종료
> ``` shell
> docker-compose down 
> # 또는
> docker compose down 
> ```
